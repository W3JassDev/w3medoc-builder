W3MEDOC — Development
Исходники и сборщик оффлайн-справочника Warcraft 3.

Суть
Этот репозиторий не предназначен для конечных пользователей. Здесь лежит движок сборки и сырой контент. Готовый результат (скомпилированный справочник) выкладывается на форумах (XGM и др.).

Технические заметки
Среда: Node.js (v24.14.1).

Логика: Сборщик берет файлы из content/, прогоняет через шаблоны и сохраняет результат в dist/.

Зависимости: Если появятся сторонние пакеты, они будут в package.json.
